Interactive Navigation Demo

Scroll down to see the navigation bar change its appearance. The navbar is fixed to the top of the page and will transition to a dark theme when you scroll. Each menu item has hover effects including a colored underline animation and color change.

The navigation is fully responsive - on mobile devices, the menu will collapse into a hamburger menu that can be toggled. The mobile menu inherits the same scroll behavior and hover effects as the desktop version.

To implement this navigation bar on other pages, simply copy the entire navigation component (nav element and its contents) along with the associated CSS and JavaScript to your other HTML files.
